# 构建Redis服务



## 1. 备份`docker-compose.yml`

<img src="assets/image-20201215205058830.png" alt="image-20201215205058830" style="zoom:50%;" />



## 2. 修改

修改`docker-compose.yml`内容如下：

```yml
version: '3'
services:
    redis:
        image: redis
        expose:
            - 6379
        ports:
            - "6399:6379"
```



## 3. 构建

运行如下命令：

```
docker-compose up
```
