# 正则表达式概述

## 思考

#### 场景1：在一个文件中，查找出itcast开头的语句

测试文件
```
itcast hello python
itcast c++
itheima ios
itheima php
```

#### 场景：在一个文件中，找到含有itcast的语句

测试文件
```
hello itcast python
www.itcast.cn c++
itheima ios
itheima php
```


#### 场景：在一个文件中，找到邮箱为163或者126的所有邮件地址



想要完成上述几个场景中的需求，可以用正则表达式非常简单的完成，这就是我们学习正则表达式的目的
