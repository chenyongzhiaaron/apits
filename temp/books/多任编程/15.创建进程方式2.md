# 创建进程方式2

## 0. 回忆线程创建的方式

```python
import threading


class Task(threading.Thread):
    def run(self):
        """子线程要执行的代码"""
        pass
    
t = Task()
t.start()
```



## 1. 示例代码

```python
from multiprocessing import Process
import time
 
class MyNewProcess(Process):
    def run(self):
        while True:
            print('---1---')
            time.sleep(1)
 
if __name__=='__mian__':
    p = MyNewProcess()
    # 调用p.start()方法，p会先去父类中寻找start()，然后在Process的start方法中调用run方法
    p.start()
 
    while True:
        print('---Main---')
        time.sleep(1)

```

## 2. 小结

1. 此种创建多进程的流程
   1. 自定义一个类，继承Process类
   2. 实现run方法
   3. 通过自定义的类，创建实例对象
   4. 调用实例对象的start方法
2. 对比
   1. 自定义继承Process类的方式比 直接创建Process对象 要稍微复杂一些，但是可以用来实现更多较为复杂逻辑的功能
3. 建议
   1. 如果想快速的实现一个进程，功能较为简单的话，可以直接创建Process的实例对象
   2. 如果想实现一个功能较为完整、逻辑较为复杂的进程，可以自定义继承Process类 来实现
