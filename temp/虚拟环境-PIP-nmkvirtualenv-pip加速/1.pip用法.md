# pip的用法

## 1. 安装一个包

#### 安装最新版

```shell
 $ pip install <package_name>
```

#### 指定版本安装

```shell
 $ pip install <package_name>==<version>
```



## 2. 升级一个包

> 如果不提供version号，升级到最新版本

```shell
 $ pip install --upgrade <package_name>==<version> 
```



## 3. 删除一个包

```shell
 $ pip uninstall <package_name>  
```
