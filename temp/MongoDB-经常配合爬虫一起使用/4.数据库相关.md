# MongoDB数据库相关命令



## 一、查看所有的数据库

```
show dbs
```

或者

```
show databases
```



## 二、查看当前正在使用的数据库

```
db
```

注意：没有切换数据库的情况下默认使用test数据库



## 三、切换数据库

```
use 数据库名字
```



## 四、创建数据库

```shell
use 数据库名字
```

注意：

* 如果数据库不存在，则创建数据库，否则切换到指定数据库
* 刚创建的数据库并不在数据库的列表中，如要显示它，需要向数据库中插入一些数据，才会看到刚刚创建的数据库（即如果创建了数据库没有数据，show dbs看不到）
* MongoDB 中默认的数据库为 test，如果你没有创建新的数据库，集合将存放在 test 数据库中



## 五、删除当前的数据库

```
db.dropDatabase()
```
